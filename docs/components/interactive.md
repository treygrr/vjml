# Interactive Components

These components own the richer email widgets: accordions, carousels, navbars, and social blocks. Their child tags are documented here alongside the parent component instead of as separate leaf pages.

## Accordion

Use accordion for compact FAQ and support content. It works with `mj-accordion-element`, `mj-accordion-title`, and `mj-accordion-text`.

## Carousel

Carousel handles image rotation, thumbnails, and Outlook fallback markup. It works with `mj-carousel-image` children.
